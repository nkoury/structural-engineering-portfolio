# Client Site Version

This is a separate publishable copy of the portfolio for a simpler GitHub Pages path.

Differences from the root site:

- No `Process` navigation tab
- No resume section or resume PDF
- Separate editable content file at `client-site/js/content.js`
- Separate content editor at `client-site/content-editor.html`

When GitHub Pages serves the repository root, this version is available under:

```text
/client-site/
```

Use `client-site/content-editor.html` to edit only this version's copy.
