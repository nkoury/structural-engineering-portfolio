# About Page Photos

Place public-ready personal photos here.

Suggested filenames:

- `noah-portrait.webp`
- `noah-field.webp`
- `noah-pursuit.webp`

After adding photos, wire them into `about.html` in place of the current photo panels.

Example:

```html
<div class="about-photo portrait">
  <img src="assets/about/noah-portrait.webp" alt="Noah Koury portrait" />
  <span>Portrait</span>
</div>
```
