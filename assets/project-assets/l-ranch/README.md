# L Ranch Project Assets

This folder is the public-safe asset folder for the L Ranch case study.

Use this folder for files that are safe to publish to GitHub Pages:

- Redacted model screenshots or preview images
- Redacted drawing crops
- Analysis screenshots or diagrams
- Calculation excerpts with sensitive information removed
- Public SwiftXR embed metadata

Do not place confidential original U3D files here unless you intentionally want them in the public repository.
Use `work/private-assets/l-ranch/u3d-source/` for local staging of original U3D files before upload/conversion.
